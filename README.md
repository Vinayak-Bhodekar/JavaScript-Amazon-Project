Amazon-Style JavaScript Project 

This project is a basic Amazon-like shopping website built using HTML, CSS, and JavaScript.
It includes core e-commerce features to practice DOM manipulation, events, and frontend logic.

Features

Product listing page

Add to cart functionality

Cart page with item count & total price

Remove items from cart

Cart data stored in localStorage

Simple checkout flow UI

Responsive layout (if included)

Technologies Used

HTML

CSS

JavaScript (DOM + LocalStorage)



Purpose

This project was created while learning frontend web development to understand how an e-commerce site works at a basic level.
